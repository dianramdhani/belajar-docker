## Panduan Penggunaan
Menggunakan file sh yang telah dibuat.
```bash
# Membuat container
sh create.sh

# Menjalankan container
sh start.sh

# Menghentikan container
sh stop.sh

# Menghapus container
sh remove.sh
```
